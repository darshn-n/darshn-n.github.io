# Inacions Repo

A new Flutter project about my Inactions.

## Installation

You can clone by copy and pasting below commands in your Terminal.

```bash
    git clone https://github.com/darshn-n/inactions-repo/
    cd inactions-repo
    flutter run
```

App Folder Structure

```
📦lib
 ┗ 📜main.dart
```
